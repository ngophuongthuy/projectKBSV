# Base by Development Department KBSEC

Source code for start new project React create by [Vite](https://vitejs.dev/)

## Installation

Use the npm or yarn to install.

```bash
npm install 
```
```bash
yarn install 
```

## Usage
Run project
```bash
npm run start 
```
```bash
yarn start 
```

Build
```bash
npm run build
```
```bash
yarn build
```
